# Orizon-OS
**A streamlined Ubuntu-based environment designed for simplicity.**

Orizon-OS is a custom desktop experience built on top of Ubuntu 24.04 LTS. It is designed to make the transition from Windows to Linux as seamless as possible for average users.

The core philosophy of the project is the **"One Week Without Terminal"** challenge: providing a GUI-driven experience so robust that a user never needs to open the command line for daily tasks.

## ✨ Key Features
- **Intuitive Interface:** A deeply customized KDE Plasma environment with a familiar, modern workflow.
- **Complete Branding:** Custom application icons, Orizon Plymouth boot screen, and unified system aesthetics.
- **Automated Setup:** Integrated Bash scripts that handle environment configuration, theming, and optimizations automatically.
- **Orizon-Kiosk Mode:** Specialized restricted environment support for specific use cases.

## 🚀 Quick Installation

To transform your standard Ubuntu installation into Orizon-OS, open your terminal one last time and run:

```bash
# Clone the repository
git clone [https://github.com/publuseba/orizon-os.git](https://github.com/publuseba/orizon-os.git)

# Enter the directory
cd orizon-os

# Make the script executable and run it
chmod +x install-kde.sh
sudo ./install-kde.sh
